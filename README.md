## 功能特性

- ✅ 播放/暂停控制
- ✅ 上一曲/下一曲切换
- ✅ 播放列表管理
- ✅ 进度条控制与跳转
- ✅ 音量控制与静音
- ✅ 单曲循环/随机播放
- ✅ 歌曲搜索功能
- ✅ 响应式设计
- ✅ 键盘快捷键支持
- ✅ 播放记录本地存储

## 技术栈

- HTML5 Audio API
- CSS3 (Flexbox, Grid, 动画)
- JavaScript (ES6+)
- 本地存储(LocalStorage)

## 快速开始

1. 克隆项目到本地
2. 准备音乐文件（MP3格式）放入 `assets/music/` 目录
3. 准备专辑封面图片放入 `assets/images/` 目录
4. 修改 `script.js` 中的播放列表数据
5. 用浏览器打开 `index.html`

## 部署方式

1. **Gitee Pages**: 上传到Gitee仓库并开启Pages服务
2. **GitHub Pages**: 上传到GitHub仓库并开启Pages服务
3. **本地运行**: 直接打开index.html文件

## 快捷键

- **空格键**: 播放/暂停
- **Ctrl + →**: 下一曲
- **Ctrl + ←**: 上一曲
- **M**: 静音切换
- **R**: 重复模式切换
- **S**: 随机播放切换
